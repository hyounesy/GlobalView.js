(inner, constant) OPTIONS :OptionDescription

A map of valid options with option descriptions, validation functions and flags about side effects

Type:
  • OptionDescription
Properties:
Name Type Description
padding OptionDescription

The space around the drawing area in the form [top, right, bottom, left]. X-axis, y-axis and colormap are drawn within padding space.
showColormap OptionDescription

When enabled, shows a colormap to the right of the plot.
enableScrolling OptionDescription

When enabled, scrolling above the plot zooms in or out of the data.
enableThumbnailDragging OptionDescription

When enabled, thumbnails can be dragged with the mouse.
enableContinuousRendering OptionDescription

When enabled, the canvas is continuously rerendered at up to 60 frames per second. Keep this setting disabled to save processing resources.
enableTransparency OptionDescription

Enables/disables blending in WebGL. Whenever using any kind of transparency, this setting should be kept enabled.
showPointDensity OptionDescription

When enabled, draws an image into the background, that shows density of points. (can be combined with 'showPointClusters')
showPointClusters OptionDescription

When enabled, draws an image into the background, that shows colored clusters of points. (can be combined with 'showPointDensity')
pointClusterThreshold OptionDescription
showXAxisHistogram OptionDescription

When enabled, shows a histogram between the x-axis and the plot.
showYAxisHistogram OptionDescription

When enabled, shows a histogram between the y-axis and the plot.
showColormapHistogram OptionDescription

When enabled, shows a histogram between the colormap and the plot.
numHistogramBins OptionDescription

Controls the number of bins within each histogram in the scatterplot.
histogramHeight OptionDescription

Controls the height of each histogram in the scatterplot (in pixels).
pointShape OptionDescription

Controls the shape of data points in the scatterplot.
customPointShape OptionDescription

When 'pointShape' is set to 'Custom', this defines a GLSL function given vec2 p, that returns opacity in the range [0.0 ... 1.0] at location p.
pointSize OptionDescription

Controls the diameter of data points in the scatterplot (in pixels).
pointOpacity OptionDescription

Controls the visibility of data points in the scatterplot between 0 (invisible) and 1 (fully opaque).
pointColor OptionDescription

Controls the color of data points in the scatterplot. Valid values are an array of bytes in RGBA order or a colormap name.
thumbnailSize OptionDescription

Controls the width/height of thumbnails in the scatterplot (in pixels).
thumbnailBorderWidth OptionDescription

Controls the width of thumbnail borders in the scatterplot.
thumbnailBorderColor OptionDescription

Controls the color of thumbnail borders in the scatterplot. Valid values are an array of bytes in RGBA order, a color name or 'null'. If set to 'null', the CSS foreground color will be used.
thumbnailLineColor OptionDescription

Controls the color of thumbnail line in the scatterplot. Valid values are an array of bytes in RGBA order, a color name or 'null'. If set to 'null', the CSS foreground color will be used.
thumbnailLabelColor OptionDescription

Controls the color of thumbnail labels in the scatterplot. Valid values are an array of bytes in RGBA order, a color name or 'null'. If set to 'null', the CSS background color will be used.
labelThumbnails OptionDescription

When enabled, links thumbnails to points using unique labels instead of lines.

onMouseDown :onMouseDownCallback

Event handler that sets flags about how to process this mouse-down event

The following properties of 'event' can be set to true:

  • pointSelection: Enable selection of data points
  • pointDragging: Enable dragging of data points
  • viewDragging: Enable moving of the view
  • lassoSelection: Enable selection of data points with a rectangular lasso
  • polygonLassoSelection: Enable selection of data points with a freeform lasso
