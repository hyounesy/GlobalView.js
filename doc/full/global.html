(constant) OPTIONS :Object

A map of valid options with option descriptions, validation functions and flags about side effects

Type:

Properties:
Name Type Description
padding Array.<number> | Array.<string> | number | string

The space around the drawing area. X-axis, y-axis and colormap are drawn within padding space. Values can be specified as array of length 4 for [top, right, bottom, left], or as a single value specifying all four corners. Values can be absolute (e.g. 120 or "120" ) or percentages (e.g. "20%"" or ["10%", "20%", "10%", "20%""] (default: [50, 60, 50, 50])
showColormap boolean

When enabled, shows a colormap to the right of the plot.
enableScrolling boolean

When enabled, scrolling above the plot zooms in or out of the data. (default: true)
enableThumbnailDragging boolean

When enabled, thumbnails can be dragged with the mouse. (default: true)
enableContinuousRendering boolean

When enabled, the canvas is continuously rerendered at up to 60 frames per second. Keep this setting disabled to save processing resources. (default: false)
enableTransparency boolean

Enables/disables blending in WebGL. Whenever using any kind of transparency, this setting should be kept enabled. (default: true)
showPointDensity boolean

When enabled, draws an image into the background, that shows density of points. (can be combined with 'showPointClusters') (default: false)
showPointClusters boolean

When enabled, draws an image into the background, that shows colored clusters of points. (can be combined with 'showPointDensity') (default: false)
pointClusterThreshold number

Specifies the density threshold ratio. Densities below threshold * maximum-density are considered outliers. (default: 0.1). See ClusterMapOptions
showXAxisHistogram boolean

When enabled, shows a histogram between the x-axis and the plot. (default: false)
showYAxisHistogram boolean

When enabled, shows a histogram between the y-axis and the plot. (default: false)
showColormapHistogram boolean

When enabled, shows a histogram between the colormap and the plot. (default: false)
numHistogramBins Object

Controls the number of bins within each histogram in the scatterplot.
histogramHeight number

Controls the height of each histogram in the scatterplot (in pixels). (default: 64)
pointShape string

Controls the shape of data points in the scatterplot. Acceptable values are 'Rectangle', 'Circle', 'Cross', 'Diamond', 'Gaussian', 'Custom'. (default: 'Circle')
customPointShape string

When 'pointShape' is set to 'Custom', this defines a GLSL function given vec2 p, that returns opacity in the range [0.0 ... 1.0] at location p. (default: '{ return 1.0; }')
pointSize number

Controls the diameter of data points in the scatterplot (in pixels). (default: 6)
pointOpacity number

Controls the visibility of data points in the scatterplot between 0 (invisible) and 1 (fully opaque). (default: 1.0)
pointColor Array.<number> | string

Controls the color of data points in the scatterplot. Valid values are an array of bytes ( 0 .. 255) in RGBA order, a color name (e.g. "red"), or a colormap name ("exhue" or "rainbow"). (default: "exhue")
thumbnailSize number

Controls the width/height of thumbnails in the scatterplot (in pixels). (default: 64)
thumbnailBorderWidth number

Controls the width of thumbnail borders in the scatterplot. (default: 1)
thumbnailBorderColor Array.<number> | string

When enabled, links thumbnails to points using unique labels instead of lines. (default: false)

(package) findRepresentativePoints(dataset, dim0, dim1, densityMap, numPointsToReturn, dist, targetRatio) → {Array.<number>}

Choose k characteristic points from the given dataset based on the given density map

This function computes point densities for each point in the dataset. It returns the first Math.floor(k * (1 - targetRatio)) lowest density points (outliers) and the first Math.floor(k * targetRatio) highest density points (cluster centers), given that all returned points are at least 'dist' apart.

Parameters:
Name Type Description
dataset Dataset
dim0 number

The first input dimension (This value must match the one used to compute the histogram)
dim1 number

The second input dimension (This value must match the one used to compute the histogram)
densityMap DensityMap
numPointsToReturn number

Maximum number of points to return
dist number

Minimum Euclidean distance between returned points in normalized space (in percentage of overall data space bounds)
targetRatio number

A ratio between 0 (only outliers) and 1 (only cluster centers)
